# f2e2nd_week1
> Practicing using Vue3 + Typescript + vue-utils(jest) to implement.

## Build Setup

```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run serve

# run unit test
npm run test:unit

# eslint format
npm run lint

# build for production with minification
npm run build
```
